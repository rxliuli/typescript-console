# TypeScript Console

Run and debug TypeScript code in the Chrome DevTools.

Key features:

- Write and run TypeScript directly in DevTools
- Instant compilation and execution
- Supports latest TypeScript features
- Easy to use: just Cmd/Ctrl+S to run
